# CentrX Color Palette - 60/30/10 Design System

## Design Philosophy
Following the proven 60-30-10 rule for UI/UX design:
- **60%**: Neutral colors for backgrounds and main content areas
- **30%**: Secondary neutral colors for cards and supporting elements  
- **10%**: Navy accent color for important actions and emphasis

## Color Distribution

### Primary Colors (60% Usage)
**Neutral Foundation - Used for backgrounds, main content areas**

- `neutralWhite` - #FAFAFA - Main backgrounds
- `neutralLight` - #F5F5F5 - Secondary backgrounds  
- `neutralGray` - #E5E5E5 - Borders, dividers
- `neutralMedium` - #9E9E9E - Placeholder text
- `neutralDark` - #424242 - Body text
- `neutralBlack` - #212121 - Headings, primary text

### Secondary Colors (30% Usage)  
**Supporting Neutrals - Used for cards, inputs, secondary content**

- `secondaryLight` - #F8F9FA - Card backgrounds
- `secondaryGray` - #DEE2E6 - Input backgrounds
- `secondaryDark` - #6C757D - Secondary text

### Accent Colors (10% Usage)
**Navy Focus - Used sparingly for important elements**

- `accentNavy` - #1E3A8A - Primary buttons, important actions
- `accentNavyLight` - #3B82F6 - Hover states
- `accentNavyDark` - #1E40AF - Active states

### Status Colors (Minimal Usage)
**Semantic Colors - Used only for status indicators**

- `successGreen` - #10B981 - Success states
- `warningOrange` - #F59E0B - Warning states  
- `errorRed` - #EF4444 - Error states
- `infoBlue` - #3B82F6 - Information states

## Usage Guidelines

### 60% - Neutral Foundation
- Main app backgrounds (`neutralWhite`)
- Content areas (`neutralLight`) 
- Text hierarchy (`neutralBlack` → `neutralDark` → `neutralMedium`)
- Borders and dividers (`neutralGray`)

### 30% - Secondary Support  
- Card containers (`secondaryLight`)
- Input fields (`secondaryGray`)
- Secondary information (`secondaryDark`)

### 10% - Navy Accent
- Primary action buttons (`accentNavy`)
- Important navigation elements
- Key interactive components
- Emphasis and focus states

## Design Benefits

✅ **Accessibility**: High contrast ratios between text and backgrounds
✅ **Consistency**: Limited color palette prevents design chaos  
✅ **Focus**: Navy accent draws attention to important elements
✅ **Scalability**: Easy to maintain and extend
✅ **Professional**: Clean, modern appearance suitable for any audience

## Color Psychology

- **Neutrals**: Create calm, professional, trustworthy feeling
- **Navy**: Conveys reliability, authority, and importance
- **Minimal Color**: Reduces cognitive load and improves usability

This palette is designed to be timeless, accessible, and suitable for college students and professionals alike.
